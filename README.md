# OtpravkaApi

## Назначение сервиса
OtpravkaApi — веб-сервис, предназначенный для управления отчетами об отправке электронных писем.

## Описание решаемой задачи
Сервис предоставляет REST API для добавления, получения, удаления и поиска отчетов об отправке писем. 
Решение может использоваться для тестирования, отладки и демонстрации работы веб-API.

## Целевая аудитория
- разработчики веб-приложений;
- тестировщики API;
- обучающиеся, изучающие ASP.NET Core Web API.

## Основные функции
- получение списка отчетов;
- добавление нового отчета об отправке письма;
- удаление всех сохранённых отчетов;
- поиск отчетов по адресу получателя.

## Архитектура и технологии
Веб-сервис реализован с использованием следующих технологий:
- ASP.NET Core Web API;
- язык программирования C#;
- REST-архитектура;
- формат обмена данными JSON.

Для документирования API используется Swagger, настройка которого выполнена в практических работах №3 и №4.

## Развёртывание
Проект развёрнут с использованием платформы Amvera.
Для сборки используется файл amvera.yml.


## Инструкция по запуску
1. Клонировать репозиторий проекта.
2. Открыть проект в среде разработки Visual Studio.
3. Убедиться, что установлен .NET SDK.
4. Запустить проект.
5. После запуска веб-сервис становится доступен по HTTP-адресу приложения.
