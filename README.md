OtpravkaApi

OtpravkaApi — это веб-сервис, разработанный на платформе ASP.NET Core Web API, предназначенный для хранения и управления отчетами об отправке электронных писем. Сервис позволяет фиксировать информацию об отправленных сообщениях и получать её в удобном виде через REST API.

Основная задача сервиса — предоставить простой и наглядный API для добавления, получения, удаления и поиска отчетов об отправке писем. Решение может использоваться в учебных целях, для тестирования и отладки REST-сервисов, а также в качестве демонстрационного примера работы веб-API.

API сервиса реализует стандартные HTTP-методы и возвращает данные в формате JSON. Для удобства разработки и тестирования ранее была подключена интерактивная документация Swagger (OpenAPI), позволяющая просматривать доступные эндпоинты и выполнять тестовые запросы напрямую из браузера.

Проект ориентирован на разработчиков веб-приложений и студентов, изучающих ASP.NET Core Web API, а также на тестировщиков, работающих с REST-интерфейсами. Структура проекта и формат документации позволяют быстро ознакомиться с функциональностью сервиса и принципами его работы.

Исходный код проекта размещён в публичном репозитории GitHub. В корне репозитория находится файл README.md с описанием сервиса, а также конфигурационные файлы, необходимые для сборки и развертывания приложения.

Проект может быть развернут локально или с использованием платформы Amvera. Для сборки и публикации используется файл конфигурации amvera.yml.
